# Publish Guide

This repository is already initialized locally.

## Suggested repository name

- `claude-code-team-starter`

Good alternatives:

- `claude-code-safe-workflows`
- `claude-code-team-kit`

## Create the remote repository

Create an empty public repository on GitHub with no README, no license, and no `.gitignore`.

## Attach the remote

```bash
git remote add origin git@github.com:Keepfox/claude-code-team-starter.git
```

If you prefer HTTPS:

```bash
git remote add origin https://github.com/Keepfox/claude-code-team-starter.git
```

## Push the existing history

```bash
git push -u origin main
```

## Recommended GitHub settings after push

- add a repository description
- add topics such as `claude-code`, `mcp`, `developer-tools`, `ai-workflows`
- enable Issues
- enable Discussions if you want community feedback
- keep the default branch as `main`

## Recommended first description

Safe, team-friendly starter repo for Claude Code with shared commands, subagents, hooks, skills, and MCP examples.
