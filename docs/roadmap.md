# Roadmap

## Near-term

- add an onboarding-focused MCP profile for team rollout
- add more stack-specific skills and hooks on top of the starter base
- publish short terminal captures or GIFs for install and first-run flows
- prepare a `v0.2.0` release with the expanded variant set

## Mid-term

- add repo templates for common stack combinations
- add a small docs site or examples gallery
- add more deterministic safety hooks with clear opt-in behavior
- add richer starter diagnostics for permissions, MCP reachability, and local setup drift

## Out of scope

- cloning Claude Code itself
- shipping proprietary prompts or private integrations
- turning this repo into a monolithic workflow framework
