# v1.0.0

Released on April 1, 2026.

## What shipped

- stable starter baseline with shared Claude Code settings, commands, agents, hooks, skills, and setup doctor flow
- named workflow variants and named bundle presets for common repo and team shapes
- scenario-based MCP profiles for onboarding, product web, release delivery, security review, data workflows, internal admin tooling, issue triage, and GitHub plus Postgres
- stack-specific and workflow-specific reusable layers for frontend-heavy repos, internal tooling, Node, Python, and Go
- bundle selector, examples gallery, and CLI bundle description mode via `--describe-bundles`
- install, onboarding, capability, config-layer, and rollout docs backed by validation and release discipline

## Why this release matters

This release marks the intended scope of the repository as complete.

The repo is now a stable public starter for teams adopting Claude Code. It gives a team a clear install path, a bundle decision path, a minimal MCP path, shared workflow assets, and enough validation and documentation to roll out the setup without guesswork.
