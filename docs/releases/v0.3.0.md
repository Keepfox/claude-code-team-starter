# v0.3.0

Released on April 1, 2026.

## What shipped

- capability matrix and config-layer docs that clarify what is included, optional, or out of scope
- named starter bundles for common team shapes
- stack-aware bundles for `node-service`, `python-service`, and `go-service`
- hook recipe library for infra protection, merge hygiene, startup context, and post-edit follow-through
- stack-specific skills and hook recipes for `Node`, `Python`, and `Go`
- install and first-run terminal demo assets for README conversion and onboarding docs
- installer output that now recommends MCP profiles, matching `CLAUDE.md` examples, starter skills, and hook recipes after bundle selection

## Why this release matters

This release moves the starter from a generic team baseline into a stronger rollout kit. A team can now choose a bundle, get stack-specific guidance immediately, and onboard into a safer workflow without stitching together examples by hand.
