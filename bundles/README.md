# Bundle Presets

Bundles are named starter presets built on top of the base install plus a small set of variants.

Use a bundle when you want a faster default for a common team shape.

Bundle manifests can also recommend:

- a stack-specific `CLAUDE.md` example
- shared starter skills
- optional hook recipes
- MCP profiles

See `docs/bundles.md` for the human-readable guide.

For faster entrypoints:

- use `docs/bundle-selector.md` to choose a preset quickly
- use `docs/examples-gallery.md` for copy-paste starter paths
- use `npm run describe-bundles` for CLI bundle descriptions
